7. JPA & Hibernate.  Java Persistence API – спецификация, описывающая, как удобно представлять данные из БД в виде Java объектов. Не является фреймворком, она только показывает, что должен уметь реальный фреймворк. А вот фреймворк уже реализует спецификацию, т.е реальная работа происходит в нем. Hibernate – фреймворк, реализующий спецификацию JPA. ORM – общее понятие для представления данных из БД в виде Java объектов.
